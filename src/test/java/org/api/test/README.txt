1. These sample test help you to test an API using the RestAssured libraries
2. The API is documented at : https://the-one-api.dev/documentation
    a. Login to the above url and register with your email address
    b. Once registration is done, you will get your APIKey
    c. Copy the APIKEY and populate the  the variable oAuthKey with it. (public static String oAuthKey = "Your API KEY HERE")
3. To run the tests using build tools e.g. maven, execute mvn clean test
4. Test can also be run using the testng.xml or using the IDE of your choice
